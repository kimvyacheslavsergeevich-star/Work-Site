# 🪪 Персональный сайт-визитка | Ким Вячеслав

Современный, быстрый и адаптивный сайт-визитка (Digital-агентство / Портфолио) веб-разработчика. Разработан на чистом коде без тяжелых сторонних фреймворков и библиотек. 

Проект полностью оптимизирован для привлечения клиентов на фриланс-бирже **FL.ru**.

## 🌟 Особенности проекта

*   **⚡️ Молниеносная скорость загрузки:** Никаких конструкторов (Tilda, WordPress) — только чистый HTML5/CSS3. Сайт мгновенно открывается на любом устройстве.
*   **📱 Адаптивная верстка:** Интерфейс безупречно отображается на экранах любых устройств (смартфоны, планшеты, ПК).
*   **🎯 Интерактивный прайс-лист:** Услуги структурированы по ценам и срокам с удобным вызовом модального окна для прямого заказа.
*   **🎨 Премиальный дизайн:** Использован стиль брутализма и минимализма с модульной сеткой и интерактивными CSS-эффектами при наведении.
*   **🔗 Прямая связь в 1 клик:** Все ссылки на Telegram и Email кликабельны и открываются в модальных окнах с плавной анимацией.

## 🛠️ Технологический стек

*   **HTML5** — семантическая разметка страницы.
*   **CSS3** — продвинутая верстка (Flexbox, CSS Grid), адаптивность и CSS-переменные (`:root`).
*   **Vanilla JS** — чистый скрипт для управления состояниями анимированного модального окна.
*   **GitHub Actions** — автоматический деплой и публикация статики.

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Поместите ваше фото или изображение в корень проекта под именем `avatar.png`.
3. Откройте файл `index.html` в любом современном браузере.
